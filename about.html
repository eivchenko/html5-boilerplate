Code Example Portal

Purpose: To be able to view and download open source software.

Technologies Used: HTML, CSS, Java Script.

Frameworks/Libraries: Bootstrap, Boilerplate, Highlight.js, JQuery.

 

Original Plan

While I was writing an essay on open source development for our class, I decided that I would like my website to be about open source software development examples. As a developer I had always used internet resources extensively to help me with my work, a programmer can't remember everything so allot of times I would browse the internet for code example to help me with my daily work. I would like to create a code example site which will contain basic source code examples in various open source languages for different purposes as well as the ability to download the source code for future use.

Examples (Grouping by language)
